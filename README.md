# Digital Addiction & Productivity Analysis Dashboard

# Project Overview 🚀
This project analyzes digital behavior patterns such as screen time, gaming, social media usage, and work/study hours to understand their impact on productivity and addiction levels.

Using Tableau, this dashboard provides interactive visual insights into how users spend their time digitally and identifies trends that may indicate digital addiction.

# Objectives 🎯
- Analyze daily screen time behavior
- Compare productive vs non-productive usage
- Identify patterns of digital addiction
- Track trends over time
- Provide actionable insights for better time management

---

# Dataset Features 📂

The dataset includes:

- User ID
- Gaming Hours 🎮
- Social Media Hours 📱
- Work/Study Hours 📚
- Total Screen Time ⏱️
- Addiction Score ⚠️
- Date (for trend analysis)

---

# Dashboard Components 📊
1. Daily Screen Time Trend 📈
- Line chart showing how screen time changes over time
- Helps detect increasing or decreasing usage patterns </br>
2. Usage Distribution 📊
- Comparison of:
    - Gaming Hours
    - Social Media Hours
    - Work/Study Hours </br>
3. Productivity vs Distraction ⚖️
- Shows balance between productive and non-productive activities </br>
4. Addiction Score Analysis 🚨
- Highlights users with high digital dependency
- Can be used to flag risky behavior </br>
5. User-Level Insights 👤
- Drill-down analysis per user
- Identify heavy users vs balanced users
